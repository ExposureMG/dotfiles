
# ☠️ ExposureMG's Dotfiles (EXSHELL)

Personal dotfiles for ZSH and KDE (Arch Linux); No plugin managers or frameworks, straight rawdogging ZSH.

## ZSH

- GNU Stow and Konsave
- ZSH with no plugin manager
- Fast Syntax Highlighting
- FZF-Tab
- Custom Agkozak Zsh Prompt
- Shitloads of Arch Linux aliases

## Arch Linux

- Proton Wayland enabled
- Chromium Wayland and Vulkan enabled (GPU Accel)
- Slightly customized Nano nanorc

## KDE

- MacOS 14 Sonoma Wallpaper Stills (AI Upscaled)
- Konsave (nobara.knsv), Plasma 6; I dont remember what i actually included.
